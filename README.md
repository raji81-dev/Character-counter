# Character Counter with HTML and CSS

A simple character counter web application built with HTML and CSS. This tool allows users to input text in a textarea and provides a live count of the characters they have typed. It is lightweight, responsive, and perfect for integrating into any project.

## Features

- **Live Character Count:** Updates in real time as the user types.
- **Customizable:** Easily modify the styles and layout using CSS.
- **Responsive Design:** Works seamlessly on both desktop and mobile devices.
